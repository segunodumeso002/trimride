# TrimRide Barber App

This repository contains the barber-facing React Native app for the TrimRide platform.

## What it includes
- Barber authentication and onboarding
- Online/offline status management
- Incoming booking request handling
- Live booking flow and status updates
- Android build support for local testing and demos

## Tech stack
- React Native
- JavaScript / TypeScript support
- React Navigation
- Socket.IO client
- Firebase messaging support

## Getting started
1. Install dependencies:
   ```bash
   npm install
   ```
2. Start Metro:
   ```bash
   npm start
   ```
3. Run on Android:
   ```bash
   npm run android
   ```

## Notes
- Keep local environment values, secrets, and Google services files out of version control.
- This repository is intended as a clean backup and working snapshot of the current barber app state.
